# Fulfillment

[![Latest Version on Packagist](https://img.shields.io/packagist/v/jeffersongoncalves/laravel-commerce-fulfillment.svg?style=flat-square)](https://packagist.org/packages/jeffersongoncalves/laravel-commerce-fulfillment) [![Total Downloads](https://img.shields.io/packagist/dt/jeffersongoncalves/laravel-commerce-fulfillment.svg?style=flat-square)](https://packagist.org/packages/jeffersongoncalves/laravel-commerce-fulfillment) [![License](https://img.shields.io/packagist/l/jeffersongoncalves/laravel-commerce-fulfillment.svg?style=flat-square)](LICENSE.md)

Commerce fulfillment module — fulfillment sets, service zones, shipping options, and fulfillments.

## Installation

```bash
composer require jeffersongoncalves/laravel-commerce-fulfillment
```

## Usage

The service provider is auto-discovered. Publish migrations and use the module models and services directly. See the [umbrella package](https://github.com/jeffersongoncalves/laravel-commerce) for the full ecosystem.
